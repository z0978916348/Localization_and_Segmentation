## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites

* Python
```sh
numpy
torch>=1.4
torchvision
matplotlib
tensorflow
tensorboard
terminaltables
pillow
tqdm
```

### Installation

1. Clone the repo
```sh
git clone https://github.com/z0978916348/Localization_and_Segmentation.git
```
2. Install python packages
```sh
pip install requirements.txt
```
